# 🎓 CECyTEM Plantel Aculco - Sitio Web Institucional

Sitio web oficial del **Colegio de Estudios Científicos y Tecnológicos del Estado de México - Plantel Aculco**, desarrollado con tecnologías modernas y estándares profesionales.

## 📋 Descripción

Portal institucional moderno, responsive y accesible que presenta la oferta educativa, instalaciones, noticias y contacto del plantel ubicado en Aculco de Espinoza, Estado de México.

## 🚀 Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos con variables
- **Bootstrap 5.3** - Framework CSS responsive
- **JavaScript ES6+** - Interactividad moderna
- **Vite** - Entorno de desarrollo rápido
- **Node.js** - Gestión de dependencias

## 📦 Instalación

```bash
# Clonar o descargar el proyecto
cd CECyTEM-ACULCO

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Previsualizar compilación
npm run preview
